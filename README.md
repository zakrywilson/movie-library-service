# Movie Library Service

## Getting Started

### Prerequisites

* Java 8  
* Maven 3  
* Docker Engine 1.13.0

## How to Run

Run the following commands:
```bash
git clone https://github.com/zakrywilson/movie-library-service.git  
cd movie-library-service/server
mvn package  
sudo docker-compose up --build
